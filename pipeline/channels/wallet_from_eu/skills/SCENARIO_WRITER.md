# Scenario Writer — Wallet from the EU

## Mandatory Structure (8 acts)

**Act 1 — HOOK (0-12s)**
Stat shock / fear loss / paradox. First sentence = unanswered question.
No intro. No "today we". Conflict first.

**Act 2 — SETUP: The Mainstream Narrative (12-60s)**
What does everyone believe? State it generously. Let the viewer nod along.
This is the setup for the flip.

**Act 3 — THE NUMBERS (60-120s)**
Drop the data that breaks Act 2. Give: number → comparison → emotional translation.
"For context, the average is X. This number is Y. That means Z."

**Act 4 — THE MECHANISM (120-240s)**
Explain WHY the number exists. Not just what — how the system works.
Use behavioral finance bias if applicable. Use a named archetype (Dave, The Bull, etc.)

**Act 5 — HISTORICAL PARALLEL (240-360s)**
Find the rhyme. "This is exactly what happened in [year] when [X]."
Minsky Moment, Nifty Fifty, Tulip Mania — use the arsenal.

**Act 6 — WHAT THIS MEANS FOR YOU (360-420s)**
Personal stakes. Make it visceral. "If you have a savings account, this affects you."
Never vague. Always specific.

**Act 7 — THE UNCOMFORTABLE TRUTH (420-480s)**
The gut punch. The thing nobody wants to say out loud.
"The system isn't broken. It's working exactly as designed. Just not for you."

**Ending — COMMENT BAIT**
Binary question. Two clear camps. 1-2 sentences max.
"Is [X] a feature or a bug? Drop your answer below."

## Script Specs
- Length: 7,000-9,000 characters strictly
- No [Music] tags, no [SFX], no sponsor reads, no ad blocks
- No named political speeches or verbatim official quotes
- Output order: VIDEO IDEA → VIDEO TITLE → FULL SCRIPT

## Pacing Model
Fast → Fast → Fast → SLOW DOWN FOR GUT PUNCH → Fast → Fast → Pause → Fast
Short sentence = emphasis. Stand-alone paragraph = maximum weight.

## Forbidden Phrases
"It's important to note" / "In conclusion" / "Some experts believe" /
"This is not financial advice" as a section / passive voice / "significantly" / "dramatically"
