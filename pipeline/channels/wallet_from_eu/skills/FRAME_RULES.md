# Frame Rules — Wallet from the EU

## When to Change Frame
New frame triggers (any of these = new frame):
1. New concept or object introduced in narration
2. Punchline lands — pause after, then new frame
3. Number revealed — show the number visually
4. Scene / location changes in the story
5. Narrator switches from describing to explaining mechanism
6. Every 12-15 seconds minimum (reset point rule)
7. Emotional shift in narration tone

## appearance_order Rule
Elements appear in this order per frame:
1. Background / base scene first
2. Primary character(s)
3. Key object or data element
4. Text label or arrow
5. Secondary characters if any

Each element in appearance_order = one intermediate PNG.
This creates the "building" animation effect.

## Frame Duration Guidelines
- Hook frames: 2-4 seconds (fast, punchy)
- Setup / explanation frames: 4-8 seconds (let concept land)
- Number reveal frames: 3-5 seconds (hold on the number)
- Gut punch frames: 5-8 seconds (slow down here)
- Ending frame: 6-10 seconds (comment bait needs to breathe)

## Scenes Per Video Section
- HOOK (0-12s): 2-3 frames
- SETUP (12-60s): 4-6 frames
- NUMBERS (60-120s): 3-5 frames (one per key data point)
- MECHANISM (120-240s): 6-10 frames
- HISTORICAL PARALLEL (240-360s): 4-6 frames
- WHAT THIS MEANS (360-420s): 3-4 frames
- UNCOMFORTABLE TRUTH (420-480s): 2-3 frames
- ENDING (480s+): 1-2 frames

## Total Frame Count Formula
Video duration in minutes × 4-5 = approximate frame count
Example: 8 minute video = 32-40 frames

## What Never Gets Its Own Frame
- Transition phrases ("which brings us to...")
- Filler narration with no new concept
- Repeated information already shown

## Canvas Specs
Width: 1920px | Height: 1080px | Background: #FFFFFF
Character scale: 0.6-0.9 relative to frame height
Text labels: bold, min 36px, black #000000
